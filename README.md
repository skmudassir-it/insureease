# InsureEase — Insurance CRM Platform

Full-stack Insurance CRM with no AI dependency. Built with React, FastAPI, MongoDB, Redis.

## Stack
- **Frontend**: React 18 + TypeScript + Vite + Tailwind CSS
- **Backend**: FastAPI + Python 3.12 + Motor (MongoDB)
- **Database**: MongoDB 7.0
- **Cache**: Redis 7.2
- **Storage**: MinIO (S3-compatible)
